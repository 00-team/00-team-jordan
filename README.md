# Vito

npm package - @00-team/vito

vito is a react video player package!

## Usage

`npm i @00-team/vito`

```tsx

import { Player, Source } from '@00-team/vito'

const source:Source = 'video.mp4'

const App = () => (
    <div>
        <Player source={source} />
    </div>
)

```
